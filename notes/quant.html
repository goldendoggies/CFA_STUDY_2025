Multiple Variable Regression and Assumptions

First steps back to getting to dev of this site. Get bootstrap to take on most of the formatting.

  • Working with multiple regression is simple. Find the independent variables that describe and affect the dependent variable
  • How do the well do the independed variables you selected describe affect the dependent variable?
    • Are they significant?
    • Do they all have the same effect? Which are stronger?
    • What affects do each independent variables have on each other?
    • Do you need to add more? Or eliminate any varaible? How would you know?
  • What types of tests would you run to determine the independent variable's usefulness?
    • F test? Model as a whole?
    • Whole model?
    • Restricted Model?
